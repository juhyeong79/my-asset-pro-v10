내 자산 PRO v10.3 통합 테스트 버전

포함 기능
1. 미국주식 가격과 수량 소수점 최대 6자리 지원
2. 국내주식 수량 정수 제한
3. 숫자 입력칸 클릭 시 기존 값 전체 선택 후 덮어쓰기
4. AI 추천 상세 팝업: 상태, 이유, 숫자 예시, 위험, 지금 할 행동
5. 종목관리 국내주식/미국주식 그룹 및 필터
6. 원화/달러 실현수익과 관리 잔액 분리
7. 입금, 출금, 달러→원화 환전, 직접 조정 내역
8. PC 백업 내보내기 → 핸드폰 백업 불러오기
9. 기존 localStorage 데이터 유지 및 구버전 거래일지 마이그레이션

PC에서 핸드폰으로
- PC 앱 설정·백업 → 백업 내보내기
- JSON 파일을 카카오톡/iCloud/Google Drive로 이동
- 핸드폰 앱 설정·백업 → 백업 불러오기

내 자산 PRO v10 — 컬러 포인트 모바일 앱

포함 기능
1단계: 아이폰/안드로이드 홈 화면 설치형 PWA
2단계: 종목관리, 거래입력, 예약관리, 거래일지
3단계: 규칙 기반 AI 추천, 추천수량, 거래/예약 자동기입, 환율 갱신,
       평균단가 계산, 손실·수익 회전 계산, 백업·복원, 오프라인 캐시

데이터 저장
- 서버나 데이터베이스를 쓰지 않습니다.
- 브라우저 localStorage에 저장됩니다.
- 백업 내보내기를 정기적으로 사용하세요.

설치 전 준비
- PWA는 HTTPS 주소가 필요합니다.
- GitHub Pages 같은 무료 정적 호스팅에 이 폴더 전체를 올린 뒤,
  아이폰 Safari에서 열고 '홈 화면에 추가'를 선택하세요.

# 내 자산 PRO v10.6.1 OCR Lite

## 경량화 내용
- 대용량 OCR 코어 및 한글/영문 학습 파일을 ZIP에서 제거했습니다.
- 앱 기본 파일은 매우 작아서 GitHub 웹 업로드가 가능합니다.
- OCR 실행 시 필요한 파일을 인터넷에서 받아 브라우저 안에서 분석합니다.
- 사진 자체를 별도 서버에 업로드하는 방식은 아닙니다.

## 설치
1. ZIP 압축을 풉니다.
2. 압축 안의 파일과 폴더를 GitHub 저장소 최상위에 업로드합니다.
3. Settings → Pages에서 배포합니다.
4. GitHub Pages 주소로 앱을 엽니다.

## OCR 테스트
1. `사진 자동입력` 탭에서 증권 화면 캡처를 선택합니다.
2. `OCR 분석 시작`을 누릅니다.
3. 첫 실행은 OCR 파일 다운로드 때문에 시간이 걸릴 수 있습니다.
4. 종목명·수량·평단·현재가를 반드시 확인한 뒤 적용합니다.

## 주의
- OCR은 베타 기능이며 화면 구성과 글자 크기에 따라 오인식할 수 있습니다.
- 첫 OCR 실행에는 인터넷 연결이 필요합니다.
- 데이터는 브라우저 로컬 저장소에 보관되므로 백업 기능을 사용하세요.
